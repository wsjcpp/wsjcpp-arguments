Helpful parser for parsing application arguments

## Installing

```
wsjcpp install "https://github.com/wsjcpp/wsjcpp-arguments:master"
```

or specify version:
```
wsjcpp install "https://github.com/wsjcpp/wsjcpp-arguments:v0.0.1"
```
